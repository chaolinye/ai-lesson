启动 server（带 API key）

llama-server -m model.Q5_K_M.gguf --host 127.0.0.1 --port 8080 -ngl all -c 8192 --api-key "sk-llama-local-key"

OpenAI-compatible 端点

端点用途OpenAI 兼容
POST /v1/chat/completions对话生成（主要入口）✅

curl 测试

curl http://127.0.0.1:8080/v1/chat/completions ^
  -H "Content-Type: application/json" ^
  -H "Authorization: Bearer sk-llama-local-key" ^
  -d "{\"model\":\"local\",\"messages\":[{\"role\":\"user\",\"content\":\"你好\"}],\"temperature\":0.7}"

关键 request body 字段

字段类型说明
messages数组对话消息，含 role/content。
temperaturefloat采样温度。
max_tokensint最多生成 token 数。
streambool是否 SSE 流式返回。
tools数组function calling 工具定义。
tool_choicestring/object"auto" / "none" / "required" 或 {type:"function",function:{name:"..."}}。
response_formatobject约束输出格式：{type:"json_object",schema:{...}}。

JSON schema 约束输出

curl http://127.0.0.1:8080/v1/chat/completions ^
  -H "Content-Type: application/json" ^
  -H "Authorization: Bearer sk-llama-local-key" ^
  -d "{\"model\":\"local\",\"messages\":[{\"role\":\"user\",\"content\":\"列出三个编程语言\"}],\"response_format\":{\"type\":\"json_object\",\"schema\":{\"type\":\"object\",\"properties\":{\"languages\":{\"type\":\"array\",\"items\":{\"type\":\"object\",\"properties\":{\"name\":{\"type\":\"string\"},\"year\":{\"type\":\"integer\"}}}}}}}}"
