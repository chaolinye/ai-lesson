llama.cpp 核心参数速查

官方来源：server README。参数默认值以最新版本为准。

模型加载

参数默认作用
-m, --model FNAME—模型 GGUF 文件路径。
-hf, --hf-repo user/model[:quant]—从 Hugging Face 下载并运行。量化可选，默认 Q4_K_M。

GPU / 加速

参数默认作用
-ngl, --n-gpu-layers Nauto将多少层 offload 到 GPU。可选数字 / auto / all。
-sm, --split-modelayer多 GPU 切分模式：none / layer / row / tensor。
-dev, --device—指定 GPU 设备。none = 不 offload。
-t, --threads N-1 (auto)CPU 线程数。GPU offload 多时仍影响采样等操作。
-fa, --flash-attnautoFlash Attention：on / off / auto。可显著降低 KV cache 显存。

上下文 / KV cache

参数默认作用
-c, --ctx-size N0 (模型默认)上下文大小。直接影响 KV cache 占用。
-ctk, --cache-type-k TYPEf16K cache 数据类型。可设为 q8_0 / q4_0 等量化格式节省显存。
-ctv, --cache-type-v TYPEf16V cache 数据类型。同上。
-kvo, --kv-offloadenabledKV cache 是否 offload 到 GPU（默认开启）。
--mlockoff将模型锁定在 RAM 中，防止 swap。
--mmap, --no-mmapenabled是否内存映射模型文件。关闭则加载更慢，但可能减少 pageout。

Batch / 并发

参数默认作用
-b, --batch-size N2048逻辑 batch 大小。影响 prompt processing 阶段的计算并行度。
-ub, --ubatch-size N512物理 micro-batch。在显存/内存限制内切分 batch。
-np, --parallel N-1 (auto)server 并行 slot 数。每个 slot 处理一个请求。
-cb, --cont-batchingenabledcontinuous batching（动态批处理）。多用户时保持高吞吐。

生成 / 采样

参数默认作用
-n, --n-predict N-1 (无限)最多生成多少 token。设为 -1 不限，-2 直到填满上下文。
--temp, --temperature N0.80温度。越高越随机，越低越确定。
--top-p N0.95Top-p nucleus 采样。只在累计概率达 p 的 token 中采样。
--top-k N40只从概率最高的 k 个 token 中采样。0 = 禁用。
--min-p N0.05Min-p 采样。忽略概率低于最高概率 × min-p 的 token。0 = 禁用。
-s, --seed SEED-1 (随机)随机种子。固定种子可复现输出。
--repeat-penalty N1.00重复惩罚。>1.0 减少重复，1.0 = 禁用。
--mirostat N0 (禁用)Mirostat 采样。1 或 2 启用，自动调整 temperature。

Server 网络

参数默认作用
--host HOST127.0.0.1监听地址。仅本机用 127.0.0.1；局域网用 0.0.0.0。
--port PORT8080HTTP 端口。
--api-key KEY—API 认证密钥。可设多个逗号分隔。
--embeddingoff仅启用 embedding 模式（节省显存）。

显存估算公式

总 VRAM ≈ 量化后权重 + KV cache × 并发 + 计算缓冲 + 驱动/系统开销

KV cache per context token ≈ 2 × hidden_size × num_layers × bytes_per_element
 (× 2 是因为 K 和 V 各一份)

示例：8B 模型，hidden=4096, layers=32, f16 KV cache (2 bytes)
 每 token KV cache = 2 × 4096 × 32 × 2 = 524,288 bytes ≈ 0.5 MB/token
 8192 上下文 → KV cache ≈ 4 GB
 每多一个并发 slot 就多一份。

Q5_K_M 8B 权重 ≈ 5.5 GB + KV cache 4 GB (8K ctx) + 2 GB 余量 ≈ 11.5 GB。16GB 显存仍有余。
