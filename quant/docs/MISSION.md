# Mission: LLM 量化（Quantization）

## Why

在自己的硬件上流畅运行大模型。理解量化的原理后，能自主选择量化方案、预估显存需求、诊断运行时的质量问题，不再只是无脑下载别人做好的量化版本。

## Success looks like

- 能用一句话解释量化「到底做了什么」，并能用手画出一张量化映射图
- 能根据自己 GPU/CPU 配置，算出某个模型量化后能否跑得动
- 能自己动手用 bitsandbytes 或 llama.cpp 量化一个模型并成功运行推理
- 遇到量化后效果变差时，能推理出是哪个环节的问题（精度选择？校准数据？group size？）

## Constraints

- 逐步深入，不急于求成
- 理论 + 动手结合
- 以本地部署为最终目标

## Out of scope

- 训练量化感知训练（QAT）的完整流程
- 硬件底层（如 GPU 架构对量化核的具体实现细节）
- 非 LLM 的量化（如 CNN 量化部署）
